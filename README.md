# GamesSysTray (FRENCH ONLY!)

![Preview screenshot](https://i.imgur.com/W2uUEKI.png)

## Fonctionnalités

### L'icône dans le "systray" de Windows
Une icône dans la "systray" de Windows qui affiche une liste de jeux et les lance en 1 click. Faites un clic droit dessus pour en afficher la liste.

### L'interface de gestion (double-click sur l'icône du systray)
Avec l'interface de gestion, vous pouvez effectuer les opérations suivantes:
* Ajouter manuellement un jeu (faire un clic droit dans une zone vierge)
* Importer directement des jeux en utilisant ses raccourcis (sur le bureau par exemple) (multi-sélection disponible) (faire un clic droit dans une zone vierge)
* Modifier une entrée (nom, chemin et arguments) (faire un clic droit sur une icône de jeu)
* Supprimer une entrée (faire un clic droit sur une icône de jeu)
